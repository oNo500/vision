# Video Summary

You are given a full transcript of a Chinese live-streaming or long-form video (直播/长视频). Produce a markdown summary in Simplified Chinese.

## 核心主题
（1-2 句，说明视频类型、主播身份、核心内容）

## 产品清单
（仅适用于带货/电商直播。每行一个产品：`- **产品名** — 卖点/价格/促销信息`。非带货视频此节留空或省略）

## 章节划分
（基于时间戳，5-8 个自然章节，每节格式：`* 起止时间 章节标题`）

## 核心观点 / 知识点
（主播分享的观点、故事、知识，bulleted list，5-10 条，排除纯带货话术）

## CTA / 行动号召
（主播要求观众做的动作，如下单、加会员、录屏等，bulleted list）

Keep tone neutral; no editorializing. Output pure markdown, no code fences.
