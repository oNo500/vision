# Vision

面向全职个人创作者的工具集，覆盖短视频与直播两条主线，核心方向：

- **情报与学习** — 竞品监控、爆款拆解、账号数据追踪、知识库沉淀
- **素材处理** — 格式转换、竖屏适配、字幕生成、批量下载
- **内容生产** — AI 脚本生成（Vertex AI）、TTS 配音
- **分发自动化** — 多平台定时发布
- **选题监控** — 热榜抓取、AI 筛选每日推送
- **直播辅助** — 录播切片、弹幕处理

## 规划路线

当前处于**工具阶段**，工具成熟后逐步接入自动化工作流（工程化阶段）。
