# 直播工具全景图

```
直播工具生态
│
├── 弹幕 / 互动数据
│   │
│   ├── 官方渠道（推荐）
│   │   ├── B站开放平台      门槛低  个人可申请  官方 WebSocket 长连接  ★ 最省心
│   │   ├── 抖音开放平台     门槛高  需企业资质  SDK v1.0.8
│   │   ├── 火山引擎 RTC     门槛中  字节系生态  抖音同源，申请相对容易
│   │   └── 快手开放平台     门槛中  需申请
│   │
│   ├── 开源逆向（不稳定）
│   │   ├── DouyinLiveWebFetcher  Python  ~1.7k  ⚠ 追着抖音 signature 更新跑
│   │   ├── DouyinBarrageGrab     C#      ~1.5k  系统代理抓 WSS
│   │   ├── UniBarrage            Go      ~85    统一多平台格式转发（推荐作为中间层）
│   │   └── [停更风险] 大量项目因 signature 算法更新而失效
│   │
│   └── TikTok 国际版
│       └── TikTokLive            Python  ~1.4k  非官方，Euler Stream 商业签名服务撑着
│
├── 推流
│   │
│   ├── 协议选型
│   │   ├── RTMP    延迟 3-5s   主播推流主流方案
│   │   ├── SRT     延迟 <2s    专业广播，弱网最优
│   │   ├── HLS     延迟 6-30s  大规模分发，自适应码率
│   │   └── WebRTC  延迟 <1s    连麦 / 双向互动首选
│   │
│   ├── 开源工具
│   │   ├── OBS Studio   免费开源   最主流，高度可定制
│   │   ├── douyin-rtmp  Python     一键获取抖音推流地址
│   │   └── looplive     Python     7×24 循环多平台同推
│   │
│   └── 商业工具
│       ├── StreamYard   $49/月   浏览器端，多嘉宾连线
│       └── Restream     $19/月   30+ 平台同时推流
│
├── 录制 / 监控
│   ├── DouyinLiveRecorder  Python  ~9.6k  40+ 平台，循环值守  ★ 最活跃
│   ├── StreamCap           Python  ~3.4k  FFmpeg 核心，含 GUI
│   └── stream-rec          Kotlin  ~1.2k  弹幕同步录制
│
├── 数据分析
│   ├── 国内
│   │   ├── 蝉妈妈    GMV 监控 / 带货达人排行 / 竞品分析
│   │   ├── 抖查查    实时热度 / 观众留存 / 账号全量数据
│   │   └── 飞瓜数据  多平台（抖音/快手/B站）
│   └── TikTok 跨境
│       ├── FastMoss  商品/达人/广告七维度
│       └── Kalodata  实时销售 / 选品趋势
│
├── AI 数字人 / 无人直播
│   ├── 闪剪       7×24 不间断，1000+ 模板，订阅制
│   ├── 创客兔     声音+形象克隆，本地部署，需 RTX 3060+
│   └── 万兴播爆   文本转语音，自动回复弹幕
│
├── 美颜 / 特效 SDK
│   ├── 腾讯特效 SDK   商业授权，Web/移动端
│   ├── 相芯 FaceUnity 商业授权，可接声网 Beauty API
│   └── ZEGO 即构美颜  按量计费，全平台
│
└── 云服务 / 底层 SDK
    ├── 腾讯云 CSS    节点丰富，价格较低
    ├── 阿里云直播    阿里生态集成好
    ├── 即构 ZEGO     延迟 600ms，万人连麦
    ├── 声网 Agora    多 CDN 融合调度
    └── 火山引擎      抖音同源，字节系生态
```

## 选型速查

```
需求                        推荐方案
─────────────────────────────────────────────────────
弹幕接入（稳定）            B站官方 WebSocket
                            抖音 → 开放平台 or 火山引擎
弹幕接入（快速验证）        UniBarrage（注意维护风险）
直播录制                    DouyinLiveRecorder + FFmpeg
多平台同推                  OBS Studio + Restream
带货数据分析（国内）        蝉妈妈 + 抖查查
带货数据分析（TikTok）      FastMoss + Kalodata
AI 无人直播                 闪剪 / 创客兔
低延迟连麦                  WebRTC（即构 / 声网）
─────────────────────────────────────────────────────
```
