# AI 辅助视频制作工具栈（2026，已用 Claude Pro）

短视频制作全链路的 AI 工具图谱，从脚本到发布，每个环节都有 AI 可以深度介入。

---

## 整体链路

```
脚本生成 → 配音 → 音乐 → 视觉素材 → 视频剪辑 → 质量审查 → 发布
    ↑                                                    ↓
    └──────────── 数据复盘 → 认知积累 ←──────────────────┘
```

> [!TIP]
> **人工介入最关键的点**：AI 生成的 Hook（前 3 秒）普遍缺乏反差感和情绪爆发点，这里值得花时间手动打磨。具体数字和真实案例也要人工替换，AI 倾向于生成模糊表述。

**推荐工作流：**
```
Perplexity AI 研究热点角度（15min）
    ↓
Claude（已订阅）生成 3 个版本脚本，带结构差异（2min）
    ↓
人工重写前 3 秒 Hook（核心介入点，AI 普遍偏弱）
    ↓
AIMIX 对照爆款结构校对节奏
```

## 七、学习成长路径

这套工具链不只是用来"执行"的，更应该是认知积累的系统。

```
阶段一（0-1 个月）数据感知
    熟读平台原生数据后台
    重点：完播率曲线中观众流失的精确时间点
    工具：抖音创作者中心 / Shortimize

阶段二（1-3 个月）爆款拆解
    每周选 5 个同赛道爆款逐帧拆解
    记录：Hook 类型 / 节奏密度 / CTA 位置
    工具：EchoTik、Shortimize（妙构已无法核实）

阶段三（3-6 个月）假设验证
    固定一个变量做 A/B 测试
    顺序：先测封面 → 再测标题 → 再测开头时长
    工具：ThumbnailTest、Vibbit

阶段四（6 个月+）系统化生产
    建立个人爆款公式
    发布前用 Go Viral 质检 + 人工对照平台安全规范
    工具：Go Viral、Marketeze
```

> [!NOTE]
> 所有案例和失败数据，目的是看透分析透，转化为可复用的知识点——而不是代入情感去焦虑。用外科医生的视角解剖，而不是病人的视角感受。参考 [如何找到真正有价值的直播经验](./how-to-find-real-insights.md)。
