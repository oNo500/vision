## 产品描述

产自河南焦作温县垆土地，铁棍山药经低温烘干、石磨研磨制成。垆土保水保肥，铁棍山药淀粉细腻、黏液蛋白丰富，冲泡即食，健脾养胃，老少皆宜。

## 产品亮点

- 原产地垆土认证，非普通沙地山药
- 低温烘干锁住黏液蛋白，营养不流失
- 石磨研磨，粉质细腻无颗粒感
- 冲泡即食，60° 水冲开即可，不结块
- 无添加无增稠剂，配料表只有山药

## FAQ

| 问题 | 回答 |
|------|------|
| 和普通山药粉有什么区别？ | 铁棍山药产自温县垆土地，黏液蛋白含量是普通山药的3倍以上，口感更细腻，养胃效果更明显。 |
| 怎么冲泡？ | 取一勺（约20g）先用少量60度温水调成糊，再加热水搅匀。不能用开水直接冲，会结块。 |
| 糖尿病人能喝吗？ | 山药GI值较低，但不做医疗建议，请在医生指导下决定。 |
| 保质期多久？ | 密封干燥存放18个月，开封后3个月内喝完，放冰箱冷藏更佳。 |
| 有没有小包装试喝？ | 拍500g正装赠送3小包独立装，出差旅行携带方便。 |
