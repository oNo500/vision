# Roadmap

面向全职个人创作者，优先解决重复体力劳动，把时间留给判断和创作。

两条主线并行：**短视频** + **直播**，工具成熟后逐步接入自动化工作流。

---

## 阶段一：素材处理自动化

> 最高频、最体力、效果立竿见影。

- [ ] ffmpeg 封装
  - 批量格式转换（mp4/mov/avi → 统一格式）
  - 竖屏适配（横屏 → 9:16，自动补黑边或裁切）
  - 批量加水印（位置/透明度可配置）
  - 批量提取封面帧（指定时间点或自动取第 1 帧）
  - 批量压缩（控制输出体积，适配各平台上传限制）
- [ ] yt-dlp 封装
  - 批量下载（支持列表/频道）
  - 无水印下载（抖音/TikTok）
  - 指定格式/分辨率下载
- [ ] Whisper 字幕
  - 视频/音频 → SRT 字幕文件
  - 批量处理目录

---

## 阶段二：内容生产提效

> 用 AI 生成初稿，人工把关 Hook 和差异化表达。

- [ ] 脚本批量生成
  - 主力：Google Vertex AI（Gemini 2.5 Flash 批量 / Gemini 2.5 Pro 高质量）
  - 输入：选题 + 时长 + 账号风格
  - 输出：多版本脚本初稿，存入脚本库
- [ ] TTS 配音
  - 优先：Azure TTS（每月 50 万字符免费，API 完善）
  - 备选：CosyVoice 3.0 本地部署（Apache-2.0，商用安全）
  - 支持声音克隆

---

## 阶段三：分发自动化

> 内容准备好之后，发布不应该占用人的时间。

- [ ] 多平台定时发布（基于 social-auto-upload）
  - 支持：抖音、小红书、视频号、Bilibili
  - 自动填写标题、标签、封面
  - 定时队列管理
- [ ] 发布结果通知（Telegram Bot）

---

## 阶段四：选题监控

> 每天早上自动推送今日选题建议，减少选题决策耗时。

- [ ] 热榜抓取
  - 抖音热点宝 / 抖音指数
  - 蝉妈妈飙升视频榜
- [ ] AI 筛选（Vertex AI Gemini 2.5 Flash）
  - 匹配账号定位和内容风格
  - 判断热点生命周期阶段
- [ ] 每日选题推送（Telegram Bot + Notion 存档）

---

## 阶段五：直播辅助

> 风险最高，最后做，优先走官方 API 和离线工具。

- [ ] 离线工具（无平台风险）
  - 录播转字幕（Whisper，复用阶段一成果）
  - 直播音频后处理（降噪、混音）
  - 直播切片：长录播自动切割为短视频素材
- [ ] 弹幕处理（走官方开放平台 API）
  - 弹幕朗读（TTS 播报）
  - 关键词触发自动回复

---

## 工程化阶段（工具稳定后）

以上工具模块成熟后，串联为端到端自动化工作流：

```
热榜监控 → 选题推送 → 脚本生成 → 配音合成 → 素材处理 → 定时发布 → 数据复盘
```

- [ ] n8n 工作流编排（自托管）
- [ ] 统一配置管理（账号信息、API Key、发布规则）
- [ ] 数据复盘自动化（完播率、互动率周报）
